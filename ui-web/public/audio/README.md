# Audio Assets

This directory contains audio files for the Pacman game.

## Required Files

| File | Description | Format |
|------|-------------|--------|
| `chomp.mp3` | Pellet eating sound | MP3 |
| `power_pellet.mp3` | Power pellet pickup | MP3 |
| `ghost_eat.mp3` | Eating ghost sound | MP3 |
| `death.mp3` | Player death sound | MP3 |
| `game_start.mp3` | Game start jingle | MP3 |
| `siren.mp3` | Ghost chase siren | MP3 |
| `retreat.mp3` | Ghost retreat sound | MP3 |
| `extra_life.mp3` | Extra life earned | MP3 |
| `background_music.mp3` | Background music | MP3 |

## Adding Audio Files

1. Place your audio files in this directory
2. Ensure files match the names above exactly
3. Recommended: Use MP3 format for broad browser support
4. Keep file sizes small (< 100KB for effects, < 2MB for music)
